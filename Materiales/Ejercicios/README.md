# Ejercicios y Resolución

Ejecución técnica de **Programación 1**. Este módulo aloja desafíos prácticos diseñados para consolidar el pensamiento computacional y la algorítmica aplicada. 

La colección está integrada por ejercicios oficiales de cátedras universitarias de referencia y propuestas de autoría propia destinadas a extender los desafíos lógicos.

---

## Codificación y Control de Calidad

Cada solución de código depositada en este directorio responde a criterios estrictos de ingeniería:
*   **Gobernanza de Estilo:** Alineación mandatoria a las especificaciones de la [PEP 8](../Links/).
*   **Análisis de Complejidad:** Documentación interna sobre la eficiencia del flujo, estructuras de datos seleccionadas y mutabilidad.
*   **Reproducibilidad:** Diseño modular mediante funciones aisladas para permitir pruebas unitarias y análisis estático limpio.

## Guía de Reproducción

Para ejecutar y auditar cualquiera de estos de código de forma aislada:

1.  Ubicarse en el directorio correspondiente desde la terminal:
```
bash
    cd Python_UTN/Materiales/Ejercicios
    ```
```
2.  Lanzar el intérprete de Python apuntando al script específico:
```
bash
    python nombre_del_ejercicio.py
    ```
```
